## Getting Started

### Requirements
- Node.js 18+

### Setup

```bash
git clone https://github.com/huamanraj/aitoolbox.software.git
cd aitoolbox.software
npm install
npm run dev
````

Open [http://localhost:3000](http://localhost:3000)
